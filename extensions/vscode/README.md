# DARKAI VS Code Extension

VS Code extension for DARKAI - AI-powered code assistant

## Features

- Real-time code analysis as you type
- Inline error highlighting
- Quick suggestions
- Language auto-detection

## Installation

1. Copy the extension folder to VS Code extensions directory
2. Restart VS Code
3. Configure API key in settings

## Configuration

Add to VS Code settings.json:

```json
{
  "darkai.apiKey": "your-api-key",
  "darkai.apiProvider": "openai",
  "darkai.enabled": true,
  "darkai.language": "ar"
}
```
